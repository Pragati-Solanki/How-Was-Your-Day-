# 🐱 How Was Your Day?

A cute little web app that asks how your day was and tries to cheer you up.

## 🌐 Live Demo

👉 https://rad-cascaron-72f093.netlify.app

## Features

- 😊 Happy cat if you had a good day
- 🦆 Dancing ducks if you had a bad day
- 💾 Remembers your last choice
- 📱 Responsive design
